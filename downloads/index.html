Download Zia


The actual offline version as a Software Application is not currently available. You can still work on the raw HTML offline

Use the Online Editor as Win/Mac/Linux App.

One requires basic Node.js and npm environment knowledge to perform this. Don't try out if you are unaware of what you are doing.

  • Download Node.js
  • Open Node.js Command Prompt
  • Type npm install nativefier -g
  • Let the Process Complete
  • Then, once the typing area is activated again, type nativefier --name "Zia Vector Graphics Editor" "https://app-ocalypse.github.io/zia/project/"
  • Then go to the Default Directory ( as C:\Users\user in windows)and find a folder named ZIA Vector Graphics Editor-win32-x64 or something alike.
  • Then go to \ZIA Vector Graphics Editor-win32-x64\resources\app and replace icon.ico with the Desired logo, preferably the one at https://raw.githubusercontent.com/SparkScratch-P/zia/742925bc9df7886e8ce85bf44b9199d696f59249/zia.svg.

If any developer can perform this and submit to us a Download link of a .zip of this app, his/her name will be credited and will be considered as a Contributor!
